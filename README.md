# Bright-Coffee-Shop-Analysis 
# Project Description

This project analyzes transactional sales data from a coffee shop to generate actionable insights that support business growth and improved product performance.

The dataset captures daily sales activity, including product details, transaction times, quantities sold, and pricing. The goal of this analysis is to identify key revenue drivers, understand customer purchasing patterns, and provide data driven recommendations to improve overall sales performance.

# Key business questions addressed in this project include:

Which products generate the most revenue
What time of day the store performs best
Sales trends across products and time intervals
How to improve revenue and operational efficiency

# Objectives
Analyze product performance and revenue contribution
Identify peak sales periods and customer behavior patterns
Evaluate high-performing and low-performing products
Provide strategic recommendations to increase revenue

# Tools & Technologies Used
SQL (Databricks / MySQL)
Used for data cleaning, transformation, and analysis, including aggregations, time based calculations and revenue metrics.
Microsoft Excel
Used for data exploration, pivot tables and intermediate analysis.
Used to create interactive dashboards and visualizations for business insights.
Miro
Used for planning data flow, architecture diagrams, and project design.
PowerPoint / Canva
Used to present insights and recommendations to stakeholders in a clear and visually appealing format.

# Key Insights
A small number of products contribute the majority of total revenue
Sales peak during specific time periods (e.g., mornings)
Certain products consistently underperform and may require optimization
Customer purchasing behavior varies across time intervals

# Recommendations
Focus on promoting and stocking top-performing products
Introduce time-based promotions during slow periods
Optimize pricing strategies for better profitability
Improve customer retention through loyalty programs

# Project Deliverables
SQL scripts for data analysis
Processed dataset (Excel/Sheets)
Dashboard visualizations (Power BI/Tableau)
Presentation slides (PowerPoint/Canva)
Data flow and architecture diagram (Miro/Figma)

# Conclusion

This project demonstrates how data analytics can be used to transform raw transactional data into meaningful business insights, enabling informed decision-making and revenue growth.
